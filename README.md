# HTML Entity Converter

[![Continuous Integration](https://github.com/djstompzone/htmlentityconverter/actions/workflows/CI_build.yml/badge.svg)](https://github.com/djstompzone/htmlentityconverter/actions/workflows/CI_build.yml)

[![License](http://img.shields.io/badge/License-Apache_2-red.svg?style=flat)](http://www.apache.org/licenses/LICENSE-2.0)

This is a plugin for Notepad++ written in C#. It allows users to convert highlighted text to/from HTML entities, and nothing else.

## Detailed Overview of Features

- ✅ Convert selected text to HTML entity/entities
- ✅ Convert HTML entity/entities to their decoded character(s)
- ❌ Absolutely anything else
