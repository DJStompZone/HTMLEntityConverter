# HTML Entity Converter

[![Continuous Integration](https://github.com/DJStompZone/HTMLEntityConverter/actions/workflows/CI_build.yml/badge.svg)](https://github.com/DJStompZone/HTMLEntityConverter/actions/workflows/CI_build.yml) [![License](http://img.shields.io/badge/License-Apache_2-red.svg?style=flat)](http://www.apache.org/licenses/LICENSE-2.0) ![GitHub Release](https://img.shields.io/github/v/release/djstompzone/htmlentityconverter?logo=github&label=latest&link=https%3A%2F%2Fgithub.com%2FDJStompZone%2FHTMLEntityConverter%2Freleases%2Flatest)


This is a plugin for Notepad++ written in C#. It allows users to convert highlighted text to/from HTML entities, and nothing else.

## Detailed Overview of Features

- ✅ Convert selected text to HTML entity/entities
- ✅ Convert HTML entity/entities to their decoded character(s)
- ❌ Absolutely anything else
